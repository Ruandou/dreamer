# 对话时间字段问题

## 问题描述

`ScriptDialogueLine` 目前只有 `character` 和 `content` 字段，没有时间字段。

```typescript
export interface ScriptDialogueLine {
  character: string
  content: string
}
```

当前设计是对话和镜头分开存储：
- `dialogues` - 纯文本对白，无时间
- `shots` - 视觉镜头描述
- `voiceSegments` - 从 dialogues 动态生成，有计算出来的时间（startTimeMs, durationMs）

## 需要讨论

1. 是否需要给 `ScriptDialogueLine` 添加时间字段？
2. 还是保持当前设计，时间在生成 voiceSegments 时动态计算？
