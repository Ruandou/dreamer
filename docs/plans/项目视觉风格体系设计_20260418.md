# 项目视觉风格体系设计

## 一、当前问题

**现状**: `Project.visualStyle` 仅为字符串数组，选项只有 4 个：

- `realistic` (真人写实)
- `cinematic` (电影风格)
- `anime` (动漫风格)
- `vintage` (复古调色)

**问题**:

1. 选项太少，无法覆盖仙侠、古装、武侠等题材
2. 没有结构化，无法指导 AI 生成提示词
3. 用户选择后不知道具体影响什么
4. 缺少与参考资料（面部/服装/场景）的关联

---

## 二、设计方案

### 核心思路：分层定义

将视觉风格分为 **4 个维度**，每个维度可选 1 个或多个值：

```
项目视觉风格 = 时代背景 + 艺术风格 + 色调氛围 + 画质等级
```

### 维度 1：时代背景 (Era)

定义故事发生的时代，决定服装、建筑、道具的视觉基准。

| 值              | 中文名   | 说明           | 关联参考                             |
| --------------- | -------- | -------------- | ------------------------------------ |
| `ancient_china` | 中国古代 | 汉朝至明朝古装 | 角色服装参考→各朝代汉服              |
| `xianxia`       | 仙侠玄幻 | 修仙世界、仙界 | 场景参考→仙侠场景、服装参考→仙侠服饰 |
| `wuxia`         | 武侠江湖 | 古代武侠世界   | 场景参考→武侠场景                    |
| `modern`        | 现代都市 | 当代城市生活   | 场景参考→现代场景                    |
| `republic`      | 民国时期 | 1912-1949      | 需新增民国服装参考                   |
| `futuristic`    | 未来科幻 | 赛博朋克、太空 | 需新增科幻参考                       |
| `custom`        | 自定义   | 混合或特殊时代 | 用户自定义描述                       |

**AI 应用**:

```
时代背景 = xianxia
→ 角色服装提示词：优先使用仙侠服饰词汇（道袍、灵蚕纱、天霞锦）
→ 场地提示词：优先使用仙侠场景词汇（仙山、云海、灵瀑）
→ 材质选择：白玉、灵木、仙石、琉璃
```

### 维度 2：艺术风格 (Art Style)

定义视觉呈现的艺术手法。

| 值                 | 中文名     | 说明             | 提示词关键词                                              |
| ------------------ | ---------- | ---------------- | --------------------------------------------------------- |
| `photorealistic`   | 超写实     | 接近真实照片     | photorealistic, ultra-detailed, realistic skin texture    |
| `cinematic`        | 电影感     | 电影级光影与构图 | cinematic lighting, film grain, anamorphic lens           |
| `stylized_realism` | 风格化写实 | 写实基础上艺术化 | stylized realism, semi-realistic, painterly               |
| `anime`            | 动漫       | 日式动漫风格     | anime style, cel shading, vibrant colors                  |
| `chinese_painting` | 中国风     | 水墨、国画风格   | chinese ink painting, watercolor, traditional chinese art |
| `dark_fantasy`     | 暗黑奇幻   | 哥特、暗黑风格   | dark fantasy, gothic, moody, dramatic shadows             |
| `ethereal`         | 空灵飘逸   | 仙气、梦幻       | ethereal, dreamy, soft glow, mystical atmosphere          |

**AI 应用**:

```
艺术风格 = cinematic
→ 角色提示词追加：cinematic lighting, film grain, 8K
→ 场地提示词追加：cinematic composition, anamorphic lens flare
→ 光线强调：chiaroscuro lighting, volumetric light
```

### 维度 3：色调氛围 (Color & Mood)

定义整体色彩倾向和情绪氛围。

| 值              | 中文名   | 说明         | 色调特征                                      |
| --------------- | -------- | ------------ | --------------------------------------------- |
| `warm`          | 暖色调   | 温馨、浪漫   | 橙黄为主，色温 3000-4000K                     |
| `cool`          | 冷色调   | 冷静、神秘   | 蓝青为主，色温 5000-6500K                     |
| `high_contrast` | 高对比   | 强烈明暗     | 深阴影+亮高光，dramatic lighting              |
| `low_contrast`  | 低对比   | 柔和、平淡   | 均匀光照，soft lighting                       |
| `desaturated`   | 低饱和   | 复古、压抑   | 色彩暗淡，muted colors                        |
| `vibrant`       | 高饱和   | 鲜艳、活泼   | 色彩鲜艳，vivid colors                        |
| `golden_hour`   | 黄金时刻 | 日出日落暖光 | golden hour lighting, warm glow, long shadows |
| `moonlight`     | 月光冷调 | 夜晚冷光     | moonlight, blue tint, cold atmosphere         |

**AI 应用**:

```
色调氛围 = golden_hour
→ 场地提示词：黄昏黄金时刻的暖色阳光，色温约3000K，长长柔和影子
→ 角色提示词：暖色侧光，营造温馨氛围
→ 光线方向：45度侧光或逆光
```

### 维度 4：画质等级 (Quality Level)

定义输出画质标准。

| 值         | 中文名 | 说明       | 提示词关键词                                       |
| ---------- | ------ | ---------- | -------------------------------------------------- |
| `standard` | 标准   | 常规画质   | high quality, detailed                             |
| `high`     | 高清   | 较高画质   | 8K UHD, highly detailed, sharp focus               |
| `cinema`   | 电影级 | 最高画质   | 8K UHD, cinematic quality, film grade, masterpiece |
| `artistic` | 艺术级 | 强调艺术感 | artistic, award-winning photography, fine art      |

**AI 应用**:

```
画质等级 = cinema
→ 所有提示词追加：8K UHD, cinematic quality, film grade, masterpiece
→ 镜头：anamorphic lens, shallow depth of field
→ 后期：color graded, professional post-processing
```

## 四、数据库设计

### 方案 A：JSON 字段（推荐）

在 `Project` 表新增 `visualStyleConfig` JSON 字段：

```prisma
model Project {
  // ... 现有字段

  visualStyle      String[] // 保留向后兼容
  visualStyleConfig Json?    // 新增：结构化配置

  // {
  //   "preset": "xianxia_cultivation",
  //   "era": "xianxia",
  //   "artStyle": ["ethereal", "stylized_realism"],
  //   "colorMood": ["vibrant", "golden_hour"],
  //   "quality": "cinema",
  //   "customKeywords": ["仙气", "飘逸"]
  // }
}
```

**优点**:

- 灵活扩展，不需改表结构
- 支持预设 + 自定义混合
- 向后兼容（保留原 visualStyle 数组）

### 方案 B：拆分字段

```prisma
model Project {
  // ... 现有字段

  era            String?  // 时代背景
  artStyle       String[] // 艺术风格
  colorMood      String[] // 色调氛围
  qualityLevel   String?  // 画质等级
  customKeywords String[] // 自定义关键词
}
```

**优点**:

- 查询更方便（如：筛选所有仙侠项目）
- 数据校验更严格

**缺点**:

- 需要迁移数据库
- 不够灵活

## 六、AI 提示词集成

### 1. 角色提示词集成

**当前**:

```typescript
const styleLine = (project.visualStyle || []).join('、')
// → "realistic, cinematic"
```

**优化后**:

```typescript
function buildVisualStylePrompt(config: VisualStyleConfig): string {
  const parts: string[] = []

  // 时代背景 → 服装词汇
  if (config.era === 'xianxia') {
    parts.push('服装采用仙侠风格：道袍、灵蚕纱、天霞锦等仙家织物')
  } else if (config.era === 'ancient_china') {
    parts.push('服装采用中国古代汉服风格：交领右衽、宽袖长摆、丝绸锦缎')
  }

  // 艺术风格 → 画质词
  if (config.artStyle.includes('cinematic')) {
    parts.push('电影级光影，anamorphic lens, film grain')
  }
  if (config.artStyle.includes('ethereal')) {
    parts.push('空灵飘逸，仙气缭绕，soft glow, mystical atmosphere')
  }

  // 色调氛围 → 光线描述
  if (config.colorMood.includes('golden_hour')) {
    parts.push('黄昏黄金时刻暖光，色温约3000K，柔和长影')
  }
  if (config.colorMood.includes('high_contrast')) {
    parts.push('强烈明暗对比，chiaroscuro lighting, dramatic shadows')
  }

  // 画质等级
  if (config.quality === 'cinema') {
    parts.push('8K UHD, cinematic quality, film grade, masterpiece')
  }

  // 自定义关键词
  if (config.customKeywords?.length) {
    parts.push(config.customKeywords.join('，'))
  }

  return parts.join('，')
}
```

**输出示例** (仙侠 + 空灵 + 黄金时刻 + 电影级):

```
服装采用仙侠风格：道袍、灵蚕纱、天霞锦等仙家织物，空灵飘逸，仙气缭绕，soft glow, mystical atmosphere，黄昏黄金时刻暖光，色温约3000K，柔和长影，8K UHD, cinematic quality, film grade, masterpiece
```

### 自动推导函数

```typescript
function deriveFromOldStyle(oldStyle: string[]): VisualStyleConfig {
  const config: VisualStyleConfig = {
    era: 'modern',
    artStyle: [],
    colorMood: [],
    quality: 'high'
  }

  if (oldStyle.includes('realistic')) {
    config.artStyle.push('photorealistic')
  }
  if (oldStyle.includes('cinematic')) {
    config.artStyle.push('cinematic')
    config.quality = 'cinema'
  }
  if (oldStyle.includes('anime')) {
    config.artStyle.push('anime')
  }
  if (oldStyle.includes('vintage')) {
    config.colorMood.push('desaturated', 'warm')
  }

  return config
}
```

   - 常用角度：平视、俯视、仰视
   - 运动方式：推、拉、摇、移

   - 调色风格：LUT 预设
   - 特效：光晕、颗粒、模糊
   - 比例：16:9, 9:16, 2.35:1

   - 自动匹配维度配置
