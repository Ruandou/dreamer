## 1. 后端 API 路由测试 (`packages/backend/tests/routes/`)

### 1.1 认证路由 `auth.test.ts`

- `POST /api/auth/register` - 成功注册、邮箱已存在、密码太短
- `POST /api/auth/login` - 成功登录、密码错误、用户不存在
- `GET /api/auth/me` - 获取当前用户信息（需 JWT）

### 1.2 项目路由 `projects.test.ts`

- `GET /api/projects` - 获取用户项目列表
- `POST /api/projects` - 创建项目（需认证）
- `GET /api/projects/:id` - 获取项目详情
- `PUT /api/projects/:id` - 更新项目
- `DELETE /api/projects/:id` - 删除项目
- 权限验证：访问他人项目应被拒绝

### 1.3 导入路由 `import.test.ts`

- `POST /api/import/preview` - 预览剧本解析结果
- `POST /api/import/script` - 导入到已有项目
- `POST /api/import/project` - 一键导入新建项目
- `GET /api/import/tasks` - 获取导入任务列表

### 1.4 任务路由 `tasks.test.ts`

- `GET /api/tasks` - 获取视频任务列表
- `POST /api/tasks/:id/cancel` - 取消任务
- `POST /api/tasks/:id/retry` - 重试任务
