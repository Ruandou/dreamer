# AI 剧本生成全面优化计划

## 🎯 目标

### **目标 1：智能质量门控**

1. **自动修正** - AI 根据审核意见自动优化大纲（最多 2 轮）
2. **用户确认** - 如果 2 轮后仍有问题，阻挡并等待用户决定

### **目标 2：智能剧本模式检测**

1. **自动识别** - 检测用户上传的是完整剧本还是创意想法
2. **模式选择** - 完整剧本 → 忠实解析，创意想法 → AI 创作
3. **内容保护** - 忠实模式下 100% 保留用户原意，不重写
4. **混合场景支持** - 前几集完整、后面缺失时，逐集智能降级

---

## 📐 工作流程

### **流程 0：智能模式检测（新增，第一步执行）**

```
用户上传内容
  ↓
全剧本级别检测
  ├─ 得分 ≥ 6/8 → 整体完整 → 逐集检测（处理混合场景）
  │   └─ 对每一集单独评分：
  │       ├─ 得分 ≥ 6 → 忠实解析（完整剧本）
  │       ├─ 得分 3-5 → 扩展生成（有大纲）
  │       └─ 得分 < 3 → AI 创作（完全缺失）
  │
  └─ 得分 < 6/8 → 整体不完整 → AI 创作模式
      └─ 进入流程 B（三阶段生成）
```

**检测策略：**

**Level 1：全剧本检测（快速判断）**

- 场景标记（第X场/scene X）- 2分
- 对话格式（引号/角色名+冒号）- 2分
- 角色说明 - 1分
- 字数 > 3000 - 1分
- 分集标记（第X集）- 2分

**Level 2：逐集检测（仅在 Level 1 得分 ≥ 6 时执行）**

- 对每一集单独评分
- 根据分数决定该集的处理模式
- 支持混合场景（部分完整、部分缺失）

**降级逻辑：**

```typescript
const overallScore = calculateOverallScore(fullScript)

if (overallScore >= 6) {
  // 可能是完整剧本，但需要逐集验证
  const episodesMode = detectEpisodesMode(fullScript)

  const hasMixedMode = episodesMode.some((ep) => ep.mode !== 'faithful-parse')

  if (hasMixedMode) {
    console.log('[detect] 检测到混合模式，启用逐集处理')
    // 例如：1-5集忠实解析，6-10集扩展，11-36集AI创作
  } else {
    console.log('[detect] 检测到完整剧本，使用忠实解析')
    mode = 'faithful-parse'
  }
} else {
  console.log('[detect] 检测到创意想法，使用 AI 创作')
  mode = 'ai-create'
}
```

### **流程 B：三阶段生成 + 质量门控（原有）**

```
阶段 1：并行生成大纲（现有）
  ↓
阶段 2：AI 总编剧审核（现有）
  ↓
审核结果判断：
  ├─ ✅ 通过 → 进入阶段 3
  └─ ❌ 发现问题：
      ├─ 第 1 轮自动修正（10-15秒/集）
      ├─ 重新审核
      ├─ 如果通过 → 进入阶段 3
      ├─ 如果未通过：
      │   ├─ 第 2 轮自动修正（10-15秒/集）
      │   ├─ 重新审核
      │   └─ 结果判断：
      │       ├─ ✅ 通过 → 进入阶段 3
      │       └─ ❌ 仍有问题 → 阻挡，等待用户确认
```

### 用户确认选项：

- **A. 重新生成大纲** - 从头开始阶段 1
- **B. 忽略问题，继续** - 使用当前大纲生成剧本
- **C. 取消任务** - 终止任务

### **Step 2: 实现大纲修正函数**

**文件：** `packages/backend/src/services/script-writer.ts`

```typescript
/**
 * 根据审核意见修正大纲
 * @param seriesSynopsis 全剧梗概
 * @param outlines 当前大纲
 * @param reviewFeedback 审核意见
 * @returns 修正后的大纲
 */
export async function reviseOutlinesBasedOnFeedback(
  seriesSynopsis: string,
  outlines: Map<number, string>,
  reviewFeedback: string,
  modelLog?: ModelCallLogContext
): Promise<Map<number, string>> {
  const deepseek = getDeepSeekClient()
  const registry = PromptRegistry.getInstance()

  // 格式化大纲列表
  const outlinesList = Array.from(outlines.entries())
    .sort((a, b) => a[0] - b[0])
    .map(([num, outline]) => `第${num}集：${outline}`)
    .join('\n\n')

  const rendered = registry.render('outline-revision', {
    seriesSynopsis,
    totalEpisodes: String(outlines.size),
    outlinesList,
    reviewFeedback
  })

  const options: DeepSeekCallOptions = {
    client: deepseek,
    model: 'deepseek-chat',
    systemPrompt: rendered.systemPrompt,
    userPrompt: rendered.userPrompt,
    temperature: 0.4,
    maxTokens: 4000,
    modelLog
  }

  const result = await callDeepSeekWithRetry(options, (content: string) => content.trim())
  const revisedText = result.content

  // 解析修正后的大纲文本，转换为 Map
  return parseOutlinesFromText(revisedText, outlines.size)
}

/**
 * 从文本中解析大纲列表
 * 期望格式：第1集：...\n第2集：...\n...
 */
function parseOutlinesFromText(text: string, expectedCount: number): Map<number, string> {
  const outlines = new Map<number, string>()

  // 匹配 "第N集：..." 格式
  const regex = /第(\d+)集[：:]\s*([\s\S]*?)(?=第\d+集[：:]|$)/g
  let match

  while ((match = regex.exec(text)) !== null) {
    const episodeNum = parseInt(match[1], 10)
    const outline = match[2].trim()
    if (episodeNum >= 1 && episodeNum <= expectedCount && outline) {
      outlines.set(episodeNum, outline)
    }
  }

  return outlines
}
```

```typescript
// 处理用户对审核结果的确认
fastify.post('/projects/:projectId/confirm-review', async (request, reply) => {
  const { projectId } = request.params as { projectId: string }
  const { action, jobId } = request.body as {
    action: 'regenerate' | 'continue' | 'cancel'
    jobId: string
  }

  const job = await pipelineRepository.findById(jobId)
  if (!job) {
    return reply.code(404).send({ error: '任务不存在' })

  switch (action) {
    case 'regenerate':
      // 重新生成大纲（重新开始阶段 1）
      await updateJob(jobId, {
        status: 'running',
        currentStep: 'outline-generation',
        progressMeta: { message: '用户选择重新生成大纲' }
      })
      // 重新运行三阶段流程（但不包含已生成的第一集）
      await runScriptBatchJob(jobId, projectId, targetEpisodes, {
        useThreePhase: true,
        skipReviewedOutlines: false
      })
      break

    case 'continue':
      // 忽略问题，继续生成剧本
      await updateJob(jobId, {
        status: 'running',
        currentStep: 'script-generation',
        progressMeta: {
          message: '用户选择忽略审核问题，继续生成',
          reviewStatus: 'ignored_by_user'
        }
      })
      // 继续阶段 3（需要重构 runScriptBatchJob 支持断点续传）
      await continueWithCurrentOutlines(jobId, projectId, targetEpisodes)
      break

    case 'cancel':
      // 取消任务
      await updateJob(jobId, {
        status: 'cancelled',
        progressMeta: { message: '用户取消任务' }
      })
      break
  }

  return { success: true, action }
})
```

## ⚠️ 注意事项

1. **大纲解析** - `parseOutlinesFromText` 需要健壮的正则表达式，处理 AI 输出的各种格式
2. **断点续传** - 用户确认后需要能继续执行，而不是从头开始
3. **审核意见长度** - 前端展示时需要格式化长文本
4. **并发控制** - 用户确认期间，任务应保持在 `needs_review` 状态
5. **降级策略** - 如果修正函数失败（如 AI 返回格式错误），应记录日志并继续下一轮

```typescript
const MAX_RETRIES = 2
let retries = 0
while (retries < MAX_RETRIES) {
  try {
    const outline = await generateEpisodeOutline(epNum, seriesTitle, seriesSynopsis, modelLogCtx)
    outlines.set(epNum, outline)
    break
```

#### **问题 #10: 记忆提取可以并行化（可选优化）**

```typescript
// 每生成 3 集后，批量提取记忆
const episodesToExtract: Array<{
  n: number
  episode: any
  script: ScriptContent
}> = []

for (let n = 1; n <= targetEpisodes; n++) {
  // ... 生成剧本

  episodesToExtract.push({ n, episode, script })

  // 每 3 集或最后一集时，批量提取
  if (episodesToExtract.length >= 3 || n === targetEpisodes) {
    await Promise.all(
      episodesToExtract.map(({ n, episode, script }) =>
        memoryService
          .extractAndSaveMemories(projectId, n, episode.id, script, {
            userId: project.userId,
            projectId,
            op: 'extract_episode_memories'
          })
          .catch((error) => {
            console.error(`[episode ${n}] 记忆提取失败:`, error)
          })
      )
    )
    episodesToExtract.length = 0
  }
}
```
