### Task 2: 创建 Prompt Builder Pattern

**优先级**: 🟠 High  
**预计工时**: 1.5 小时  
**风险**: 低  
**涉及文件**:

- `seedance-optimizer.ts` (enhancePrompt, generateFirstFramePrompt)
- `scene-prompt.ts` (buildScenePrompt)
- `episode-service.ts` (buildScenePrompt)
- `storyboard/prompt-builder.ts`

#### 当前问题

重复的 prompt 构建逻辑散落在多个文件中：

```typescript
// ❌ 重复模式：数组 + filter(Boolean) + join
const parts = [scriptTitle, scene.location, scene.timeOfDay, ...].filter(Boolean)
return parts.join(', ')
```

#### 目标方案

**创建通用 Builder**:

```typescript
// services/prompts/prompt-builder.ts
export class PromptBuilder {
  private parts: string[] = []
  private separator: string

  constructor(separator: string = '，') {
    this.separator = separator
  }

  add(text: string | undefined | null): this {
    if (text) this.parts.push(text)
    return this
  }

  addCondition(condition: boolean, text: string): this {
    if (condition) this.parts.push(text)
    return this
  }

  addList(items: string[] | undefined, joiner: string = '，'): this {
    if (items?.length) this.parts.push(items.join(joiner))
    return this
  }

  build(): string {
    return this.parts.join(this.separator)
  }
}
```

**使用示例**:

```typescript
// Before
const parts = [
  scriptTitle,
  scene.location,
  scene.timeOfDay,
  scene.description,
  scene.actions?.join(' ') ?? '',
  scene.dialogues?.map((d) => `${d.character}: ${d.content}`).join(' ') ?? ''
].filter(Boolean)
return parts.join(', ')

// After
return new PromptBuilder(', ')
  .add(scriptTitle)
  .add(scene.location)
  .add(scene.timeOfDay)
  .add(scene.description)
  .addList(scene.actions, ' ')
  .addList(
    scene.dialogues?.map((d) => `${d.character}: ${d.content}`),
    ' '
  )
  .build()
```

### Task 4: 改进错误处理 - 增加上下文

**优先级**: 🟡 Medium  
**预计工时**: 1 小时  
**风险**: 低

#### 当前问题

多个 catch 块缺少上下文信息：

```typescript
// ❌ 缺少上下文
catch (error) {
  console.error('Script expansion failed:', error)
}

catch (error) {
  console.error('Failed to build memory context, falling back:', error)
}
```

#### 目标方案

**创建结构化错误日志工具**:

```typescript
// lib/error-logger.ts
export function logError(
  context: string,
  error: unknown,
  metadata?: Record<string, unknown>
): void {
  const errorObj = error instanceof Error ? error.message : String(error)
  console.error(`[${context}] ${errorObj}`, {
    timestamp: new Date().toISOString(),
    ...metadata
  })
}
```

**使用示例**:

```typescript
// Before
catch (error) {
  console.error('Script expansion failed:', error)
}

// After
catch (error) {
  logError('ScriptExpansion', error, {
    episodeId,
    projectId,
    operation: 'expand_episode_script'
  })
}
```

#### 实施步骤

**Step 1**: 创建 `lib/error-logger.ts`

**Step 3**: 确保所有错误日志包含:
