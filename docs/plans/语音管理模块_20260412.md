# 语音管理模块开发计划

> **版本**：v1.0
> **日期**：2026-04-12
> **目标**：在分镜生成阶段同步产出结构化语音配置，支持多 TTS 平台（阿里云/火山引擎）音色映射

## 概述

实现 Dreamer 语音管理模块，在分镜生成阶段同步产出结构化语音配置，支持前期通过 `voiceConfig` 快速上线，并为未来音色克隆预留 `voiceId` 接口。

## 1. Prisma Schema 扩展

**文件**: `packages/backend/prisma/schema.prisma`

### 1.1 新增 VoiceSegment 表

```prisma
model VoiceSegment {
  id               String   @id @default(cuid())
  segmentId        String
  segment          Segment  @relation(fields: [segmentId], references: [id], onDelete: Cascade)
  characterId      String
  character        Character @relation(fields: [characterId], references: [id])
  
  order            Int       // 同一分镜内语音播放顺序
  startTimeMs      Int       // 相对于 Segment 开始时间的偏移(毫秒)
  durationMs       Int       // 语音时长(毫秒)
  text             String    // 台词文本
  voiceConfig      Json      // 音色配置
  emotion          String?   // 情绪标签 confused/excited/sad/neutral
  
  createdAt        DateTime @default(now())
  updatedAt        DateTime @updatedAt

  @@index([segmentId])
  @@index([characterId])
}
```

### 1.2 Character 表扩展

在 Character 表添加：

```prisma
// 语音管理字段
voiceId     String?  // 预留：音色克隆 ID（如 ElevenLabs voice_id）
voiceConfig Json?    // 可选：角色全局默认音色，作为 fallback
```

## 6. Seedance 2.0 音频参数对接

**文件**: `packages/backend/src/services/seedance-audio.ts` (新建)

### 6.1 功能说明

当调用 Seedance 2.0 并启用原生 TTS 时，从数据库读取 VoiceSegment 并组装成音频参数。

### 6.2 核心函数

```typescript
import { prisma } from '../index.js'
import type { VoiceSegment, Character } from '@dreamer/shared/types'

export interface SeedanceAudioSegment {
  character_tag: string      // @Character1, @Character2 等
  text: string              // 台词文本
  voice_config: VoiceConfig  // 音色配置
  start_time: number        // 开始时间（秒）
  duration: number          // 时长（秒）
}

export interface SeedanceAudioPayload {
  type: 'tts'
  segments: SeedanceAudioSegment[]
}

/**
 * 从数据库读取 VoiceSegment，组装成 Seedance 音频参数
 */
export async function buildSeedanceAudio(
  segmentId: string
): Promise<SeedanceAudioPayload> {
  const voiceSegments = await prisma.voiceSegment.findMany({
    where: { segmentId },
    orderBy: { order: 'asc' },
    include: {
      character: true  // 包含角色信息用于生成 character_tag
    }
  })

  const audioSegments = voiceSegments.map((vs, idx) => ({
    character_tag: `@Character${idx + 1}`,
    text: vs.text,
    voice_config: vs.voiceConfig as VoiceConfig,
    start_time: vs.startTimeMs / 1000,
    duration: vs.durationMs / 1000
  }))

  return { type: 'tts', segments: audioSegments }
}

/**
 * 获取分镜关联的所有语音片段（带角色信息）
 */
export async function getSegmentVoiceSegments(segmentId: string): Promise<Array<VoiceSegment & { character: Character }>> {
  return prisma.voiceSegment.findMany({
    where: { segmentId },
    orderBy: { order: 'asc' },
    include: { character: true }
  }) as any
}
```
