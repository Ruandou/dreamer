## 二、空间层次结构

### 1. 前景 (Foreground)

**作用**: 引导视线，增加画面深度
**元素**:

- 室内：桌面物品、花瓶、书籍、台灯
- 室外：树木、花朵、栅栏、路灯、车辆
- 建筑：门廊、台阶、花坛、招牌

### 2. 中景 (Midground)

**作用**: 画面主体，视觉焦点
**元素**:

- 室内：沙发区、餐桌、床、工作区
- 室外：建筑主体、广场中心、湖面
- 建筑：主立面、入口、窗户排列

### 3. 背景 (Background)

**作用**: 营造氛围，提供环境信息
**元素**:

- 室内：窗户外的景色、墙面装饰、远处空间
- 室外：远山、天际线、云层、其他建筑
- 建筑：周围环境、天空、远景

## 四、光影设计

### 1. 光源类型

#### 自然光

- **日光**: 直射阳光，强烈明亮
- **散射光**: 云层过滤，柔和均匀
- **黄金时刻**: 日出/日落前后，暖色调，长影子
- **蓝色时刻**: 日出前/日落后，冷色调，宁静
- **窗光**: 从窗户射入，方向性强

#### 人工光

- **现代灯具**: 吸顶灯、吊灯、筒灯、射灯、台灯、壁灯、地灯、霓虹灯
- **古代灯具**:
  - **蜡烛**: 红烛、白烛、烛台（铜/瓷/木）、烛光摇曳
  - **油灯**: 豆灯、宫灯、风灯、省油灯、暖黄光晕
  - **灯笼**: 纸灯笼、纱灯、羊角灯、走马灯、红绸灯笼
  - **火把**: 松明子、火炬、跳跃火焰、烟雾
  - **香炉**: 檀香、沉香、青烟袅袅、香烟缭绕
  - **长明灯**: 佛前灯、墓室灯、永不熄灭

### 2. 光线方向

- **顺光**: 光线从正面照射，明亮但缺乏立体感
- **侧光**: 光线从侧面照射，强调纹理和立体感
- **逆光**: 光线从背后照射，形成剪影或光晕
- **顶光**: 光线从上方照射，戏剧性强
- **底光**: 光线从下方照射，诡异、恐怖感

### 3. 光线质量

#### 硬光 (Hard Light)

- **特征**: 阴影清晰锐利，对比强烈
- **来源**: 直射阳光、聚光灯、裸露灯泡
- **效果**: 戏剧性、紧张、力量感
- **适用**: 动作场景、悬疑、反派出场

#### 柔光 (Soft Light)

- **特征**: 阴影柔和模糊，过渡平缓
- **来源**: 散射光、柔光箱、窗帘过滤
- **效果**: 温馨、浪漫、平静
- **适用**: 爱情场景、日常生活、温馨时刻

### 4. 色温 (Color Temperature)

- **暖光 (2700K-3000K)**: 黄色调，温馨舒适，家庭、咖啡厅
- **中性光 (3500K-4500K)**: 白色调，自然清晰，办公室、商店
- **冷光 (5000K-6500K)**: 蓝色调，冷静严肃，医院、办公室、科技感

### 5. 光影氛围

- **高调光 (High-Key)**: 明亮均匀，阴影少，轻松愉快
- **低调光 (Low-Key)**: 对比强烈，阴影深重，神秘紧张
- **明暗对照 (Chiaroscuro)**: 光与影强烈对比，戏剧张力
- **体积光 (Volumetric)**: 光线穿过尘埃/雾气，形成光柱

## 五、构图与视角

### 1. 景别

#### 广角全景 (Wide Shot)

- **作用**: 展现整体空间，建立环境认知
- **适用**: 第一次进入场景，展示空间规模
- **镜头**: 16-24mm 广角镜头

#### 中景 (Medium Shot)

- **作用**: 聚焦特定区域，展现细节
- **适用**: 展示功能区，如沙发区、餐桌
- **镜头**: 35-50mm 标准镜头

#### 特写 (Close-Up)

- **作用**: 强调细节，营造氛围
- **适用**: 装饰物、纹理、小物件
- **镜头**: 85mm+ 长焦镜头

### 2. 拍摄角度

- **平视**: 人眼高度，自然真实，最常用
- **俯视**: 从上往下，展现布局，空间感强
- **仰视**: 从下往上，强调高度，压迫感
- **倾斜**: 荷兰角，不安、紧张、动态

### 3. 构图法则

#### 三分法 (Rule of Thirds)

- 将画面分为3x3网格
- 重要元素放在交叉点上
- 营造平衡、自然的视觉效果

#### 引导线 (Leading Lines)

- **直线**: 走廊、道路、栏杆
- **曲线**: 河流、小路、螺旋楼梯
- **汇聚线**: 铁轨、长廊、天花板横梁
- **作用**: 引导视线到视觉焦点

#### 对称构图 (Symmetry)

- **特点**: 左右或上下对称
- **效果**: 庄重、稳定、正式
- **适用**: 古典建筑、大厅、教堂

#### 框架式构图 (Framing)

- **元素**: 门框、窗户、拱门
- **作用**: 聚焦视线，增加层次
- **效果**: 画中画，深度感

#### 前景构图 (Foreground Interest)

- **作用**: 增加画面深度
- **元素**: 植物、家具、装饰物
- **效果**: 立体感、沉浸感

## 七、场景描写模板

### 模板结构（四段式）

```
【空间与环境】核心建筑结构、标志性物体、材质与布局
【光影与氛围】光线方向、硬/柔、色温、情绪氛围
【构图与视角】景别、角度、引导线
【风格与画质】项目视觉风格、画质词
```

### 示例 1：咖啡厅（日间）

```
挑高3米的复古咖啡厅空间。红砖墙面搭配深色原木横梁，天花板悬挂三盏黄铜工业风吊灯，地面为做旧木地板。前景是吧台区域，中景可见两排皮质沙发卡座，背景是整面落地窗与窗外街景。下午45度侧光透过百叶窗投射条纹光影，与吊灯的2700K暖光交织，营造温馨慵懒的氛围。广角全景平视构图，引导线从吧台延伸至落地窗，展现空间纵深。电影质感，8K超高清，浅景深突出中景沙发区，volumetric lighting增强光柱效果。
```

### 示例 2：审讯室（低调光）

```
封闭的会谈室空间，约15平方米。双面玻璃隔断墙面，冷灰色涂料，地面为深色防滑地砖。房间中央一张金属审讯桌，两侧各一把固定椅子。天花板嵌入式LED灯发出4000K冷白光，光线均匀但压抑。广角平视构图，对称布局强调压迫感。写实电影风格，8K超高清，低调光照，阴影深重，营造紧张严肃氛围。
```

### 示例 3：山顶夜景

```
城市边缘的山顶观景台。前景是粗糙的岩石地面和低矮护栏，中景是观景平台与几棵松树，背景是远处城市天际线与万家灯火。夜空深邃，月光柔和洒在山石上，城市灯光形成温暖的橙色光晕。广角俯拍构图，引导线从前景岩石延伸至远方城市，展现壮阔空间层次。电影级夜景摄影，8K超高清，长时间曝光效果，星光点点，volumetric moonlight穿透薄雾。
```

### 示例 4：现代客厅（黄金时刻）

```
开放式现代客厅，双层挑高空间。前景是浅灰色L型布艺沙发与圆形大理石茶几，中景为整面落地窗前的用餐区（胡桃木长桌配六把餐椅），背景是开放式厨房（白色橱柜+中岛台）。地面通铺浅橡木人字拼木地板，墙面纯白色乳胶漆。黄昏黄金时刻的暖色阳光从落地窗斜射入内，在地板上投下长长的柔和影子，色温约3000K，营造温馨家庭氛围。广角平视构图，引导线从沙发经餐桌延伸至厨房，展现空间通透感。北欧极简风格，电影质感，8K超高清，浅景深虚化背景厨房区。
```

### 示例 5：古代书房（文人雅士）

```
雅致的明代书房，约20平方米。前景是一张黄花梨画案，上置端砚、湖笔、宣纸、青瓷笔筒，中景为整墙书架（线装书、卷轴、古玩），背景是雕花木窗与窗外竹林。地面铺青砖，墙面白色石灰。清晨柔和光线从木格窗斜射入内，在书案上投下斑驳竹影，色温约4000K，营造清雅淡泊的文人氛围。中景平视构图，引导线从画案经书架延伸至窗户，展现书香气息。中式古典风格，电影质感，8K超高清，浅景深突出书案文房四宝，volumetric light穿透窗纸。
```

### 示例 6：古代客栈（武侠风）

```
宽敞的两层木质客栈大堂。前景是四张八仙桌与长条凳，桌上酒壶碗筷散落，中景为柜台与通往二楼的木楼梯，背景是门口悬挂的酒旗与门外街道。地面为磨得发亮的青石板，墙面裸露木结构。黄昏时分暖色烛光从桌上油灯与墙上火把发出，色温约2700K，光影摇曳，营造江湖烟火气息。广角平视构图，引导线从前景酒桌经楼梯延伸至门口，展现空间纵深。武侠写实风格，电影质感，8K超高清，低调光照，阴影深重，强调明暗对比。
```

### 示例 7：宫廷大殿（皇家气派）

```
宏伟的宫殿正殿，面阔九间，进深五间。前景是三层汉白玉须弥座台基与蟠龙金柱（直径一人合抱），中景为九龙金漆宝座与背后的龙纹屏风，背景是殿门外天空与丹陛。地面铺设金砖（细腻光滑），墙面朱红大漆。正午阳光从殿门直射入内，照亮宝座区域，大殿深处沉入阴影，形成强烈明暗对比，营造庄严肃穆的皇家氛围。广角仰视构图，对称布局强调皇权威严，引导线从台基经宝座延伸至殿顶藻井。宫廷写实风格，电影质感，8K超高清，chiaroscuro lighting增强戏剧张力。
```

### 示例 8：古代园林（江南水乡）

```
精致的苏州园林庭院。前景是太湖石假山与一池碧水，中景为曲径通幽的回廊与六角亭，背景是白墙黛瓦的厢房与月亮门。地面铺设青石板与碎石拼花小径，墙面白色石灰粉刷。午后柔和散射光（阴天），光线均匀柔和，水面倒映亭台楼阁，色温约5000K，营造宁静雅致的园林氛围。广角俯拍构图，引导线从假山经曲径延伸至亭子，展现园林层次。江南园林风格，电影质感，8K超高清，浅景深虚化背景厢房，水面reflection增强空间感。
```

### 示例 9：仙山秘境（修仙门派）

```
悬浮于云海之上的仙山主峰。前景是白玉栏杆与盘旋而上的石阶（两侧古松参天），中景为宏伟的凌霄殿（琉璃瓦屋顶散发七彩光芒，蟠龙金柱环绕），背景是远处连绵的悬浮山峰与飞流直下的灵瀑。地面铺设五彩灵石板，散发淡淡灵光。清晨霞光从东方照射，云海翻腾呈现金色边缘，殿前仙鹤翱翔，营造缥缈仙境氛围。广角仰视构图，引导线从石阶经凌霄殿延伸至最高峰，展现仙山气势。仙侠奇幻风格，电影质感，8K超高清，volumetric lighting穿透云雾，bloom effect增强灵石光芒。
```

### 示例 10：炼丹房（修仙内景）

```
宽敞的圆形炼丹密室。前景是八卦形炼丹炉（青铜材质，炉口三昧真火呈紫色），中景为四周墙壁的灵药柜（抽屉散发微光，摆放千年灵芝、万年人参），背景是墙壁上的炼丹术符文阵法（金色发光）。地面刻画聚灵阵（蓝色光纹流转），墙面为灵玉石砌成（散发温润光芒）。室内无自然光源，主要依靠丹火与阵法光芒，色温偏暖（丹火橙紫，阵法蓝金），营造神秘修仙氛围。中景平视构图，对称布局强调炼丹炉为中心，引导线从地面阵法经丹炉延伸至墙壁符文。仙侠写实风格，电影质感，8K超高清，低调光照，glow effect突出阵法与丹火。
```

### 示例 11：剑修试剑台（仙侠武道）

```
悬崖边的巨型石质平台，约百米见方。前景是插满飞剑的剑冢（万剑朝天才立，剑气冲霄形成可见光柱），中景为试剑台中央的巨型剑碑（高十丈，刻有上古剑诀，散发凌厉剑意），背景是万丈深渊与对面山峰。地面为青石铺就，布满剑痕与剑气刻印。正午阳光直射，剑气折射出彩虹光晕，石碑周围空气因剑意而扭曲，营造凌厉威严的剑修氛围。广角俯视构图，引导线从剑冢经剑碑延伸至悬崖，展现剑修圣地气势。仙侠武侠风格，电影质感，8K超高清，高对比度，motion blur表现剑气流动，lens flare增强阳光效果。
```

### 示例 12：天庭瑶池（仙界盛景）

```
天宫后苑的仙境池苑。前景是瑶池碧水（水面漂浮七彩莲花，锦鲤散发金光），中景为池畔的蟠桃树（万年蟠桃，果实饱满散发霞光，树干缠绕祥云），背景是远处的瑶池仙宫（白玉建筑，琉璃瓦顶，仙娥飞舞）。地面铺设羊脂白玉，栏杆为汉白玉雕龙。天空呈现五彩祥云，紫气东来，霞光万道从云层射出，仙鹤与凤凰翱翔其间，营造至高无上的仙界氛围。广角全景构图，引导线从瑶池经蟠桃树延伸至仙宫，展现天庭气象。中国神话风格，电影质感，8K超高清，HDR高动态范围，bloom效果增强霞光与仙气，god rays穿透云层。
```

## 八、AI 绘画提示词技巧

### 1. 关键词优先级

**高优先级**（必须包含）:

- 空间类型（咖啡厅、卧室、街道）
- 时间/光线（黄金时刻、夜晚、阴天）
- 视角（广角、平视、俯视）

**中优先级**（增强细节）:

- 材质（红砖、木地板、玻璃）
- 家具/物体（沙发、吊灯、树木）
- 氛围词（温馨、神秘、压抑）

**低优先级**（锦上添花）:

- 画质词（8K、电影质感、超高清）
- 镜头效果（浅景深、volumetric lighting）
- 风格词（北欧、工业、古典）

### 2. 负面提示词（避免AI生成）

```
人物、动物、影子、文字、水印、logo、变形、模糊、过曝、死黑
```

### 3. 句式结构

**推荐格式**:

```
[空间描述]。[材质与物体]。[光影描述]，营造[氛围]。[构图与视角]。[风格与画质]。
```

**要点**:

- 使用句号分隔，意合为一段
- 控制在 120 字以内
- 先写实体，再写光影，最后写构图
- 避免抽象词，使用具体名词
