# 角色服装描写参考

## 一、各朝代汉服特征

### 1. 汉朝服饰（端庄古朴）

#### 男子

- **深衣**: 上下连属、交领右衽、宽袖长摆、腰间系带
- **袍服**: 直裾袍、曲裾袍、交领、长袖、及地
- **常服**: 短衣长裤、便于行动、劳动者穿着
- **首服**: 冠（官员）、巾（文人）、帻（平民）
- **颜色**: 黑、红、白为主，青、紫为贵
- **材质**: 丝、绢、麻、葛

#### 女子

- **襦裙**: 上襦下裙、交领或对襟、高腰或齐腰
- **曲裾深衣**: 衣襟缠绕、层叠飘逸、贵族女子
- **直裾深衣**: 简洁端庄、日常穿着
- **发饰**: 步摇、簪、钗、华胜
- **颜色**: 红、绿、紫、白、青
- **纹样**: 云纹、龙凤、花卉、几何

### 2. 唐朝服饰（华丽开放）

#### 男子

- **圆领袍**: 圆领窄袖、便于骑射、官员常服
- **襕袍**: 下摆加横襕、文士儒雅
- **半臂**: 短袖外衣、套穿于长衫外
- **幞头**: 黑色软帽、两脚下垂或上翘
- **颜色**: 紫（三品以上）、绯（五品以上）、绿、青
- **材质**: 绫、罗、绸、缎、锦

#### 女子

- **襦裙装**: 短襦长裙、高腰至胸、披帛飘带
- **坦领**: 领口开阔、展现脖颈、开放风气
- **大袖衫**: 宽大飘逸、套穿于襦裙外
- **胡服**: 翻领窄袖、条纹裤、波斯风格
- **颜色**: 红、黄、绿、紫、金，鲜艳夺目
- **纹样**: 宝相花、联珠纹、团花、鸟兽
- **发饰**: 高髻、花钿、步摇、金簪、梳篦

### 3. 宋朝服饰（清雅简约）

#### 男子

- **直裰**: 直身长袍、交领宽袖、文士常服
- **襕衫**: 圆领或交领、下摆有襕、科举士子
- **道服**: 宽松舒适、居家隐士、淡雅色系
- **东坡巾**: 方形高顶、文人标志
- **颜色**: 白、青、灰、褐，素雅为主
- **材质**: 棉、麻、素绢

#### 女子

- **褙子**: 对襟长衫、两侧开衩、套穿于襦裙外
- **襦裙**: 窄袖短襦、瘦长裙、含蓄内敛
- **大袖礼服**: 祭祀婚礼、庄重华丽
- **颜色**: 淡粉、浅绿、月白、藕荷、鹅黄
- **纹样**: 折枝花、小碎花、暗纹、素面
- **发饰**: 低髻、简约簪钗、冠梳

### 4. 明朝服饰（端庄规整）

#### 男子

- **道袍**: 交领大袖、两侧开衩、内有摆、文士标配
- **直身**: 简洁直筒、日常穿着
- **圆领袍**: 官服、补子标识品级（文禽武兽）
- **网巾**: 束发用、黑色网状、成年男子
- **四方平定巾**: 方形软帽、儒生标志
- **颜色**: 红（官服）、青、绿、白、黑
- **材质**: 绸、缎、纱、罗、棉

#### 女子

- **袄裙**: 上袄下裙、交领或立领、马面裙
- **马面裙**: 前后光面、两侧打褶、裙门绣花
- **披风**: 长款对襟、外出保暖、贵族女子
- **霞帔**: 礼服配饰、刺绣精美、命妇专用
- **颜色**: 红（婚礼）、绿、紫、蓝、月白
- **纹样**: 吉祥图案（福寿喜）、花鸟、龙凤
- **发饰**: 狄髻（假髻）、头面（整套首饰）、挑心

## 二、仙侠服饰体系

### 1. 基础款式

#### 道袍类

- **直裰道袍**: 交领右衽、宽袖长摆、腰间丝绦、素色为主
- **八卦道袍**: 衣摆绣八卦图、深蓝或黑色、道士标配
- **云纹道袍**: 衣身流云暗纹、月白或青色、修仙者常服
- **鹤氅**: 对襟大袖、外罩长袍、仙风道骨、白色或灰色

#### 裙装类

- **留仙裙**: 多层裙摆、轻盈飘逸、仙女标配
- **百褶裙**: 细褶密集、行动如水、少女灵动
- **月华裙**: 每褶一色、五彩渐变、光华流转
- **云水裙**: 淡蓝或白色、水波纹暗纹、水系修士

#### 外衣类

- **披风**: 长款、带帽、刺绣云纹或仙鹤、防风保暖
- **大袖衫**: 外罩、半透明纱质、仙气飘飘
- **鹤氅**: 白色长袍、仙家标志、高修穿着
- **霞帔**: 仙女礼服、五彩霞光、华丽庄重

### 2. 等级标识

#### 入门弟子

- **颜色**: 灰、白、浅蓝、淡青
- **材质**: 粗布、麻衣、素绢
- **纹样**: 无或简单门派标志
- **配饰**: 基础玉牌、普通丝绦

#### 内门弟子

- **颜色**: 蓝、青、白、月白
- **材质**: 丝绸、锦缎、灵蚕纱
- **纹样**: 门派标志、云纹、简单阵法
- **配饰**: 精製玉牌、储物袋、下品法器

#### 核心弟子/长老

- **颜色**: 紫、金、白、玄色
- **材质**: 天蚕丝、灵锦、仙纱
- **纹样**: 复杂阵法、龙凤、祥云、莲花
- **配饰**: 高阶玉牌、本命法宝、仙器

#### 掌门/大能

- **颜色**: 金、紫、白、七彩
- **材质**: 仙家织物、天材地宝织就
- **纹样**: 天地大道、太极、先天八卦
- **配饰**: 仙器、法宝、仙冠、仙履

### 3. 职业特色

#### 剑修

- **款式**: 窄袖劲装、便于拔剑、紧身利落
- **颜色**: 白、黑、青、银
- **材质**: 坚韧灵布、防御加持
- **特征**: 腰间佩剑、剑穗飘摇、衣摆剑纹
- **配饰**: 剑鞘、剑穗、储物戒指

#### 丹修

- **款式**: 宽松道袍、多口袋装药材
- **颜色**: 棕、褐、绿、黄
- **材质**: 耐火灵布、防丹火
- **特征**: 腰间葫芦（装丹药）、药囊、袖口耐火纹
- **配饰**: 炼丹手套、药铲、丹方卷

#### 阵修

- **款式**: 长袍大袖、便于刻画阵法
- **颜色**: 蓝、紫、黑、金
- **材质**: 阵纹灵布、自带简易阵法
- **特征**: 衣袖绣阵图、腰间阵盘、手持阵旗
- **配饰**: 阵旗、阵盘、罗盘、阵纹笔

#### 御兽师

- **款式**: 劲装短打、便于行动
- **颜色**: 棕、绿、黄、兽纹
- **材质**: 兽皮、坚韧灵布
- **特征**: 兽纹装饰、御兽铃、腰间兽牌
- **配饰**: 御兽铃、兽粮袋、契约卷轴

#### 法修（术士）

- **款式**: 宽袖长袍、施法飘逸
- **颜色**: 蓝、紫、白、金
- **材质**: 法阵灵布、灵力传导
- **特征**: 衣袖符箓纹、腰间符袋、手持法器
- **配饰**: 符箓袋、法杖、拂尘、八卦镜

## 三、服装材质分类

### 1. 凡间材质

#### 平民布料

- **麻布**: 粗糙耐磨、灰色或本色、劳动者
- **葛布**: 夏日清凉、粗糙、底层百姓
- **棉布**: 柔软舒适、明清普及、日常穿着

#### 贵族丝绸

- **绢**: 细腻光滑、轻薄、白色或淡色
- **绫**: 斜纹光泽、轻薄柔软、高档
- **罗**: 透气孔眼、夏季礼服、轻盈
- **绸**: 平滑光亮、中等厚度、常用
- **缎**: 正面光滑、华丽、礼服用
- **锦**: 多彩提花、最昂贵、皇家贵族

### 2. 仙侠材质

#### 灵兽材料

- **灵蚕丝**: 天蚕吐丝、柔软坚韧、灵力传导
- **冰蚕丝**: 冰蚕所吐、冰凉防暑、白色透明
- **火蚕丝**: 火蚕所吐、温暖御寒、红色或金色
- **龙鳞布**: 龙鳞编织、防御极强、金色鳞片纹
- **凤羽纱**: 凤凰羽毛织就、轻盈发光、五彩

#### 灵草材料

- **灵麻**: 灵草纤维、比凡麻柔软、淡绿色
- **云棉**: 云端灵草、洁白如雪、保暖轻便
- **藤甲**: 灵藤编织、柔韧防御、绿色或棕色

#### 仙家织物

- **天霞锦**: 朝霞织就、五彩流光、仙家礼服
- **云雾纱**: 云雾凝聚、半透明、仙气飘飘
- **星辰缎**: 星光织入、夜间发光、深蓝底色
- **太极绸**: 黑白太极纹、大道至理、高人穿着
- **八卦布**: 八卦阵纹、阵法加持、阵修专用

## 四、颜色象征意义

### 1. 五行色系

- **木（青/绿）**: 生机、治愈、木系修士、春
- **火（红/紫）**: 热情、攻击、火系修士、夏
- **土（黄/棕）**: 稳重、防御、土系修士、长夏
- **金（白/银）**: 刚毅、剑修、金系修士、秋
- **水（黑/蓝）**: 深邃、智慧、水系修士、冬

### 2. 等级色彩

- **白色**: 纯洁、仙气、高修、剑修
- **青色**: 自然、平和、道门、木系
- **蓝色**: 深沉、法修、水系、阵修
- **紫色**: 尊贵、高阶、雷系、长老
- **金色**: 至高、仙帝、佛修、神圣
- **黑色**: 神秘、魔修、暗系、低调
- **红色**: 热情、火系、喜庆、危险

### 3. 特殊色彩

- **月白**: 淡蓝白、清雅、文人、仙家
- **藕荷**: 淡紫粉、温柔、女子、春季
- **黛色**: 深青黑、沉稳、老者、秋冬
- **朱砂**: 鲜红、辟邪、符箓、道教
- **玄色**: 黑中泛红、庄重、礼服、天道

## 五、配饰与细节

### 1. 头部配饰

#### 男子

- **冠**: 玉冠（贵族）、木冠（文人）、金冠（帝王）
- **巾**: 东坡巾（文人）、逍遥巾（隐士）、纶巾（军师）
- **簪**: 玉簪（束发）、木簪（朴素）、金簪（华丽）
- **发带**: 丝质、颜色多样、飘逸

#### 女子

- **步摇**: 走动摇晃、珠玉垂坠、华丽
- **簪钗**: 金簪、玉钗、银钗、固定发髻
- **花钿**: 额头装饰、花形、金箔或彩纸
- **华胜**: 大型头饰、节日礼服、凤凰牡丹
- **珠花**: 珍珠串成、精致小巧、日常

### 2. 腰部配饰

- **丝绦**: 丝质腰带、打结或玉环固定
- **玉佩**: 身份象征、温润君子、叮噹作响
- **储物袋**: 修仙必备、装物品、空间法宝
- **荷包**: 绣花小袋、装香料或丹药
- **葫芦**: 丹修标志、装丹药或灵酒
- **剑穗**: 剑修专属、飘带装饰、剑柄悬挂

### 3. 手持物品

- **拂尘**: 道家标志、白色马尾、仙风道骨
- **折扇**: 文人雅士、题字画画、夏季
- **玉箫/笛**: 音修法宝、音律攻击、雅致
- **书卷**: 儒修标志、经书典籍、文雅
- **八卦镜**: 阵修或道士、辟邪侦查
- **法器**: 剑、刀、戟、珠、钟、鼎等

## 六、服装描写公式

### 基础公式

```
颜色 + 材质 + 纹样 + 款式 + 配饰 + 气质
```

### 示例

#### 清冷剑修

```
月白色灵蚕纱道袍，衣摆绣银色流云暗纹，交领右衽，宽袖长摆，腰间系青色丝绦悬挂一柄长剑。外罩半透明白色鹤氅，仙气飘飘。头戴白玉冠，黑发如瀑。整体气质清冷出尘，凌厉中带着仙风道骨。
```

#### 温婉仙女

```
淡粉色留仙裙，外层轻纱如雾，内层丝绸贴身，裙摆层层叠叠如莲花绽放。上衣为月白对襟褙子，袖口绣折枝花纹。腰间系淡绿色丝绦，悬挂一枚羊脂玉佩。发髻低挽，插一支粉色步摇，走动时珠玉轻响。气质温婉娴静，如春风拂面。
```

#### 威严掌门

```
紫金八卦道袍，衣身以天霞锦织就，金色太极图绣于胸前，紫色祥云纹遍布衣摆。外罩黑色镶金边鹤氅，领口袖口皆绣先天八卦。头戴紫金冠，腰悬掌门玉牌与本命仙剑。脚踏云履，步履间灵光流转。整体气质威严庄重，不怒自威，仙风道骨中透着大道至理。
```

#### 灵动少女

```
鹅黄色百褶裙，每褶细密如水波，行动时裙摆摇曳生姿。上衣为浅绿对襟短襦，袖口窄小便于活动，胸前绣蝴蝶戏花图案。腰间系粉色丝绦，悬挂一个绣花荷包与小巧储物袋。头发双鬟，各插一支银质蝴蝶簪。气质灵动活泼，如林间小鹿。
```

## 九、AI 绘画服装提示词技巧

### 关键要素

**必须包含**:

- 款式名称（道袍、襦裙、劲装）
- 颜色（月白、朱砂、玄色）
- 材质（灵蚕纱、天霞锦、云雾纱）
- 纹样（流云暗纹、八卦图、折枝花）

**增强细节**:

- 穿着效果（飘逸、垂坠、贴身）
